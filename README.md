#  Beneficios de GitHub para estudiantes

- En primer lugar me parecen una gran ventaja para los estudiantes ya que tienen a disposición un amplio set de herramientas que les ayuda en sus proyectos personales o grupales y nos abre muchos caminos para poder avanzar y crecer como estudiantes.

- Las herramientas que más me llamo la atención es **Intro to Web Dev**, porque no solo te enseña a crear tu propio sitio web, sino que aprendes a como diseñarlo mientras aprendes conceptos básicos del desarrollo web y el como eso te puede ayudar en futuros proyectos.
